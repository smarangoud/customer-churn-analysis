# Customer Churn Analysis

## Objective

Analyze customer data to identify factors contributing to churn and provide actionable business insights.

## Tools Used

* Python (Pandas)
* SQL
* Matplotlib / Seaborn

## Steps Performed

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Customer segmentation using SQL
* Visualization of churn patterns

## Key Insights

* Customers with month-to-month contracts have higher churn rates
* Customers with low tenure are more likely to churn
* Higher monthly charges are associated with increased churn

## Business Recommendations

* Promote long-term contracts with discounts
* Improve onboarding experience for new customers
* Provide targeted retention strategies for high-risk customers

## Dataset

Telco Customer Churn Dataset (Kaggle)
